# 极·鬼剑术（暴风式）

## 职业稳定语义

多刃圆形或同心阵列围绕清楚中心组织多方向运动，并以中心汇聚或全阵释放形成高潮。设计稿中的具体数量只作为待核验线索。

## 职业通用 Prompt

```text
a circular multi-blade formation with concentric ring organization, coordinated blade motion around a clear center, a distinct formation-wide culmination, retain the documented count only when confirmed by source inventory
```

## 源资源约束

具体剑数、浮空方式、阵列层、人物、武器和中心特效均待 manifest/inventory 核验。不能为满足设计数量改变原阵位、索引或新增对象。

## 阶段验收

应能读出“阵列建立或呈现 -> 多刃运动 -> 中心或全阵高潮”。具体剑数、布阵时序、循环、命中和收尾待 manifest/inventory 核验。
