# 猛龙断空斩 - Vergil（维吉尔）暗蓝幻影主题

## 职业基础

引用 `../../prompts/猛龙断空斩.md`。

## 主题增量 Prompt

```text
Vergil phase-dash treatment, fractured-space afterimages, cobalt blink particles, icy motion-blur gaps, cyan rift sparks, sparse phantom-sword barrage accents
```

## 具体变化

既有高速移动路径用碎裂空间残影和闪烁粒子表达，端点或命中处点缀青色裂纹火花与少量幻影剑。

## 主题验收

每个实际存在的移动与端点节点保持可辨，裂空残影表达高速位移但不能抹平连续阶段。

## 主题排除

排除连续雾带粘连全程、密集圆形剑阵、红橙火花，以及凭主题另造瞬移阶段。
