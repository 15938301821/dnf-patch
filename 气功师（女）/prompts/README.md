# 女气功职业通用 Prompt 索引

## 一、职责

本目录定义不随主题变化的职业语义：运动、轮廓、阶段、层次、锚点和人物/特效边界。真实 NPK、IMG 和帧映射只从 `../manifest.json` 与实包 inventory 获取。

## 二、加载顺序

1. 读取源帧与 manifest，确认资源分类、阶段、几何和显示名映射状态。
2. 只有显示名映射已核验时，才读取本目录同名职业 Prompt。
3. 只有同一映射已核验时，才读取所选主题的同名 Prompt；否则只使用 manifest 独立授权的主题共同规则。
4. 回灌时继承源尺寸、Canvas、坐标、锚点、alpha 和帧序。

## 三、稳定结构

每个文件固定包含：`职业稳定语义`、`职业通用 Prompt`、`源资源约束`、`阶段验收`。英文代码块可直接组合给绘图模型；中文字段负责稳定路由和验收。

透明背景、无角色、无背景、居中和固定画布都不是职业级硬规则，只能在 manifest 已确认适用时使用。
