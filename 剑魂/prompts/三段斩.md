# 三段斩

## 职业稳定语义

三次连续冲刺斩击形成三条可区分的横向弧形轨迹，并在每次命中处给出独立冲击响应。三段之间要有清楚的空间与时序间隔。

## 职业通用 Prompt

```text
three consecutive dash slashes, three readable horizontal arc trails, a distinct impact wave for each strike, clear spacing and temporal separation between all three segments
```

## 源资源约束

三段的真实帧组、方向与人物/特效分层待 manifest/inventory 核验。不得为满足次数新增帧或改变原位移锚点、人物姿态与武器关系。

## 阶段验收

三段顺序、三条轨迹和三次命中反馈必须清楚，不能压成单次爆发。每段实际起止帧及整体收尾待 manifest/inventory 核验。
