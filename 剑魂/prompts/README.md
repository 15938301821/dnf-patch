# 剑魂职业 Prompt 索引

## 职责

本目录保存主题无关的动作、轮廓、阶段、层次和命中辨识。这里的 Prompt 不保存主题色、主题材质或资源映射。

## 加载顺序

先读取仓库根规则、`../AGENTS.md` 与经核验的 manifest/inventory。只有技术资源和显示名映射已核验时，才读取本目录同名 Prompt，并追加主题目录下的同名 Prompt；映射未核验时不得按名称路由逐技能 Prompt。

## 稳定结构

每个文件固定使用“职业稳定语义、职业通用 Prompt、源资源约束、阶段验收”四段。英文代码块用于与主题增量组合，中文段落用于边界和验收。
