# 三段斩 - Vergil（维吉尔）暗蓝幻影主题

## 职业基础

引用 `../../prompts/三段斩.md`。

## 主题增量 Prompt

```text
Stinger-inspired cobalt dash traces, a distinct cold-blue shockwave accent on each inherited strike, sharp luminous edges, sparse directional neon particle spray
```

## 具体变化

每个既有连斩段落配独立冰蓝冲击波和冲刺残迹，少量冷霓虹粒子沿各段运动方向喷散。

## 主题验收

三段节奏仍能逐段辨认，每段冲击波独立，不得合并为单次爆炸或遮没横向刃轨。

## 主题排除

排除暖色火花、连续雾幕、无方向粒子，以及把多段动作压成单段的表现。
