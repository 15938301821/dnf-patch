# 流心:跃 - Vergil（维吉尔）暗蓝幻影主题

## 职业基础

引用 `../../prompts/流心：跃.md`。

## 主题增量 Prompt

```text
Vergil-inspired evasive-slash styling, a cold-blue airborne afterimage, sparse floating Mirage Edge shards, azure rim light, controlled directional motion blur
```

## 具体变化

空中斩残影冷蓝化，以少量幻影剑碎片辅助轨迹，并用青蓝边光收束人物或刃轨轮廓。

## 主题验收

空中方向与斩击弧线仍清楚，碎片始终是次级层，运动模糊不能遮住动作转换。

## 主题排除

排除密集剑群、地面裂纹主导、过量运动模糊，以及强行改变人物存在性的构图。
