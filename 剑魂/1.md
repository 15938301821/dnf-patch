## 🛠️ 制作流程（维吉尔风专属）

### Step 1：导出原帧

- 打开 **ExtractorSharp** → 加载 `sprite_character_swordman_effect_xxx.NPK`
- 找到目标 `.img` → 导出全部帧 PNG（记录原尺寸，通常 256×256 或 512×512）

### Step 2：AI 出图

- 用上方对应技能 Prompt
- **SD 推荐参数**：DPM++ 2M Karras / Steps 30-40 / CFG 5-7
- **ControlNet**：Canny 锁原帧轮廓，只重绘光色 + 裂纹 + 浮游剑

### Step 3：PS 对齐

- 新帧尺寸 / 锚点必须与原帧一致
- IMG Ver5 新手可先转 Ver2 再改

### Step 4：回灌封包

- ExtractorSharp 导入替换 → 另存为 `%sprite_character_swordman_effect_vergil_blue.npk`

### Step 5：进游戏测试

- 丢入 `ImagePacks2` → 启动 DNF → 验证特效

## 💡 维吉尔风格进阶技巧

### 🔵 空间裂纹（灵魂元素）

所有技能 Prompt 均可追加：

```
spatial fracture lines, shattered glass rift, dimensional slit
```

### 🔵 幻影剑统一

- 所有浮游剑保持 **Mirage Edge 冰蓝色**
- 避免混入金色 / 红色，保持一致性

### 🔵 分层渲染建议

| 层   | 内容                | 合成模式       |
| ---- | ------------------- | -------------- |
| 底层 | 空间裂纹 / 次元裂隙 | 屏幕 / Screen  |
| 中层 | 幻影剑 / 刀刃       | 正常 / Normal  |
| 顶层 | 冰蓝辉光 / 粒子     | 叠加 / Overlay |

### 🔵 招牌技能优先级

1. **拔刀斩（Judgment Cut）** → 辨识度最高，必做精品
2. **破空斩（Judgment Cut End）** → 终极必杀，视觉冲击力最强
3. **幻影剑舞（Summoned Swords）** → 维吉尔标志性无限剑制
